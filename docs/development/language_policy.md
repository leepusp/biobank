# Project language policy

English is the official language for the Biobank codebase.

This applies to:

- Python comments
- docstrings
- Django messages
- `help_text`
- `verbose_name` and `verbose_name_plural`
- template labels
- UI buttons
- commit messages
- migration names
- developer documentation

Existing Portuguese text should be translated incrementally when a file is touched for functional changes. Avoid large translation-only patches unless they are explicitly planned.
