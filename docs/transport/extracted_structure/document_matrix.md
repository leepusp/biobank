# Shipment document matrix

| Condition | Required documents | Required labels | System action |
|---|---|---|---|
| NB-1 and non-OGM | Content declaration; external package identification | Fragile; sender/recipient identification; shipment QR | Public submission, internal review, receipt, intake |
| OGM | Content declaration; sender declaration; OGM/CIBio notification; external package identification | Fragile; biohazard; restricted opening; shipment QR; UN3373 if CR-2/NB-2 or higher | Require OGM-specific form section and internal authorization review |
| NB-2 or CR-2 | Content declaration; sender declaration; external package identification; triple packaging checklist | Fragile; biohazard; UN3373; restricted opening; shipment QR | Require Category B / UN3373 workflow |
| International shipment with genetic heritage | MTA/TTM; shipment invoice; content declaration; external package identification | According to classification | Require SisGen/MTA review before dispatch/acceptance |
| Internal CEPID B3 transfer | Shipment record; checklist; QR; optional content declaration | Internal QR; package identification | Internal request, approval, dispatch, receipt |
